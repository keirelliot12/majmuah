FITUR YANG TERSEDIA:
---------------------------------------------------------

1. AUTO-INSTALL SAAT CONTAINER START
   - Flutter SDK (stable version)
   - System dependencies (GTK, build tools, graphics)
   - Project dependencies dari pubspec.yaml
   - VS Code extensions
   - Linux desktop support

2. QUICK COMMANDS
   
   Script:
   ./flutter-commands.sh run       # Run app
   ./flutter-commands.sh build     # Build release
   ./flutter-commands.sh test      # Run tests
   ./flutter-commands.sh doctor    # Flutter doctor
   
   Make:
   make run                        # Run app
   make build                      # Build release
   make test                       # Run tests
   make clean                      # Clean project
   
   VS Code Tasks:
   Ctrl+Shift+P → "Tasks: Run Task"
   
3. DEBUG CONFIGURATIONS
   - F5 untuk start debugging
   - Profile mode
   - Release mode

CARA MENGGUNAKAN:
---------------------------------------------------------

PERTAMA KALI (GitHub Codespaces):
1. Buka repository di GitHub
2. Klik "Code" → "Create codespace on branch"
3. Tunggu container build (5-10 menit)
4. Semua dependensi otomatis terinstall!
5. Mulai coding!

PERTAMA KALI (VS Code Dev Containers):
1. Install extension "Dev Containers"
2. Buka folder project
3. Ctrl+Shift+P → "Dev Containers: Reopen in Container"
4. Tunggu setup selesai
5. Mulai coding!

RESTART/REBUILD CONTAINER:
- Semua akan otomatis terinstall ulang
- Tidak perlu manual setup lagi
- Flutter SDK dan dependencies sudah siap
